## 简介

当你在B站专栏里复制 `实际内容` 时，实际上复制出来的是:

```
实际内容 作者：XXX https://www.bilibili.com/read/cv00000000 出处：bilibili
```

这个脚本可以帮你去除这些水印。复制的是啥就是啥。
